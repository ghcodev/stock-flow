name: frontend-design
description: Refatorar UI com qualidade production-grade.
---
Ao refatorar design:
- Alterações cirúrgicas, nunca reescrever componente inteiro
- Começar pelos tokens (CSS variables) antes de componentes
- Uma página por vez com confirmação entre cada uma
- Nunca substituir SVG por bibliotecas externas
- Mostrar diff antes de salvar qualquer arquivo
