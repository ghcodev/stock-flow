---
name: adversarial-review
description: Revisar código de forma crítica após alterações grandes.
---
Após alterações grandes (5+ arquivos):
- Revisar o diff como um desenvolvedor sênior cético
- Listar o que pode quebrar
- Identificar regressões visuais
- Apontar o que foi além do escopo pedido
- Não fazer alterações — apenas reportar
